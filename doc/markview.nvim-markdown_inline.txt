------------------------------------------------------------------------------
enable

  ‚Ä¢ type: `boolean`
    default: `true`

Shows inline markdown in previews.

------------------------------------------------------------------------------
block_references

  ‚Ä¢ type: `markview.config.markdown_inline.block_refs`[^3]
    default[^4]

Changes how block references are shown.

>lua
    block_references = {
        enable = true,

        default = {
            icon = "Û∞ø® ",

            hl = "MarkviewPalette6Fg",
            file_hl = "MarkviewPalette0Fg",
        },
    },
<

enable ~

  ‚Ä¢ type: `boolean`
    default: `true`

Self-explanatory.
default

  ‚Ä¢ type: `markview.config.__inline`[^5]
    default[^6]

Default configuration for block references. See how inline elements are
configured[^7].
[string]

  ‚Ä¢ type: `markview.config.__inline`[^8]

Configuration for block references whose text matches `string`. See how inline
elements are configured[^9].

‚ñå üìú Note

  The structure for block references is `[[<text>]]` when `text` contains
  `#^`.

------------------------------------------------------------------------------
checkboxes

  ‚Ä¢ type: `markview.config.markdown_inline.checkboxes`[^10]
    default[^11]

Changes how checkboxes are shown.

enable ~

  ‚Ä¢ type: `boolean`
    default: `true`

Self-explanatory.

checked ~

  ‚Ä¢ type: `markview.config.markdown_inline.checkboxes.opts`[^12]

>lua
    checked = {
        text = "Û∞ó†",
        hl = "MarkviewCheckboxChecked",

        scope_hl = "MarkviewCheckboxChecked"
    },
<
text

  ‚Ä¢ type: `string`

Text to show as the checkbox in the preview.
hl

  ‚Ä¢ type: `string`

Highlight group for `text`.
scope_hl

  ‚Ä¢ type: `string`

Highlight group used on the list item text.

unchecked ~

  ‚Ä¢ type: `markview.config.markdown_inline.checkboxes.opts`[^14]

Configuration for unchecked checkboxes. Same structure as `checked`.

[string] ~

  ‚Ä¢ type: `markview.config.markdown_inline.checkboxes.opts`[^16]

Configuration for `[string]` checkboxes. Same structure as `checked`.
